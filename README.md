Cada main*.py representa una versió del projecte.
La versió amb número més alt (mainX.py) és la versió final i estable.

Dataset
El dataset utilitzat és MovieLens Small, inclòs dins la carpeta /dataset/.
Cal descomprimir el fitxer .zip abans d’executar qualsevol script

Per executar la versió final:
python3 mainX.py
